Hardware Page

This is my harware page and this page talks about what makes a computer and how these parts affect the compter.

Mother Board

This piece of hardware is called a motherboard.It contains the principal components of a computer or any other device.

CPU

This is the CPU is known as the brain of the computer. This is where most calculations occur

RAM

This here is RAM, stands for random access memory.This
is one of the most common type of memory found in computers

Hard Drive

The hard drive is the primary storage unit for a computer

BUS

BUS is a communication system that transfers data between
components inside a computer, or between computers.

Keyboard

A device which uses an arrangement of buttons or keys to act as a electronic switch.

Computer Mouse

A mouse is an input device that is most often used with a personal computer.

Monitor

A display screen used to provide visual output from a computer
