Vindinium

What is vindinium?

Vindium is a game that is played by programming an AI. The player must program
their bots to complete three tasks in order to win: Obtain mines, Go to Taverns and
to Attack Enemy AIs.

How I coded my AI.

Tactic explanation:

I decided to program my bot to retreat from whatever danger it is in when its health is lower than 50 to go to the closest tavern. The next thing my bot is programmed to do is to go to the closest Free Mine and make it his own. The third task my bot is programmed to do is to attack the closest enemy bot.

How my Bot decides what to do:

The code to the right is an if else statement setting
requirments for the bot. My Bot's first priority is
to make sure it's health is above 50 if the health is
lower than 50 it will perfrom the task "Tavern".
If my bot's Health is higher than 50 it will run the
task "Freemines", unless there is an enemy bot
which then it will run "closest Enemy".

How My Bot goes to Taverns
at less than 50 hp:

Looking at the code here it is simply looking at the
positions of all the taverns and determines which one is closer to my Bot wherever it is.Then console.logs that it id heading towards the closest Tavern. This is simply how it does it. The matter of when it does this is in the if else statement above. 

                    
if (task === "tavern") {
        var closestTavern = bot.taverns[0];
        for(var i = 0; i < bot.taverns.length; i++) {
            if(bot.findDistance(myPos, closestTavern) > bot.findDistance(myPos, bot.taverns[i])) {
