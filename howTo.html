How To Code:
Character ino Phaser

Step One:

The first step into programming your character
is to go on and make a sprite Sheet. A sprite
sheet is a png of multiple different frames that
have slight changes inbetween and when you
run them together the character will be
animated and seem to be doing an action like
running or jumping.

Step Two:

The next thing you're going to do is upload
your sprite sheet to your phaser project.
Once you upload your sprite sheet to the
project go into the preload function and
type: game.load.image('spriteName',
'FolderName/SpriteSheet.png');

Final Step:

The last thing you need to do is go into the
create function and add the sprite by writing:
this.Sprite = game.add.sprite(0, 0, 'spriteName);
this.sprite.animations.add('spriteName', [0, 1], 6, true);
