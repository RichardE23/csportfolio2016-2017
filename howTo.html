How To Code a To Do List

HTML: 

                    
<form id="myform">
    <div>
        <div>
            <h2>Add item:</h2>
        </div>
        <input type="text" id="newitem" />
        <select id="options">
            <option value="none">None</option>
            <option value="underline">Underline</option>
            <option value="italics">Italics</option>
            <option value="highlight">Highlight</option>
        </select>
        <input type="submit" />
        
        <br><br>
        <h2><div id="todolist"></div></h2>
    </div>
    
    </form>
  </section>

In order to code a to do list you will need to code a form, divs, and some different types of inputs. The first input you will want to code is a text input so the user is allowed to type what they need to put on their list. The next thing you will need to code is a select tag with an id and inside that tag you will code the users options. The next and last thing you will need to code is to code another input but it will be a button so the user can submit what they wrote with the option they chose.

JavaScript: 

                    
var todo = document.getElementById('todolist');
var form = document.getElementById('myform');
var field = document.getElementById('newitem');
var options = document.getElementById('options');
var newitem = document.getElementById('newitem');
                        
form.addEventListener('submit', function(evt) {
  var text = field.value;


  if (options.value == "none") {
    todo.innerHTML += '<div>' + text + '</div>';
  }
  else if (options.value == "underline") {
    todo.innerHTML += '<div style="text-decoration:underline">' + text + '</div>';
  }
  else if(options.value == "italics") {
    todo.innerHTML += '<div style="font-style: italic">' + text + '<div>';
  }
  else if(options.value == "highlight") {
    todo.innerHTML += ' <div style="color: lime">' + text + ' <div>';
  }

  evt.preventDefault();
}, false);

todo.addEventListener('click', function(evt) {
  var tar = evt.target;
  tar.parentNode.removeChild(tar);
  evt.preventDefault();
}, false);

Now that we are coding in Javascript you will need to get the divs in which you gave an id by using "document.getElementById('nameOfDiv')". Once you pull your divs into JavaScript you will need to make an addEvenListner listening for submit. Next you will create if/if else statements for the options with the condition being if the value equal to a specific option on the list. Inside that if statement you will need to code the line to display the users input. Doing so you will code a div inside the first div add style to it writing in HTML to customize the output.
