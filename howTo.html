How To Code A Flippable Image

Image to the right does not function properly!
Example is on the About Me Page. 

                    
<script src="https://code.jquery.com/jquery-2.1.4.min.js"></script>
<script src="https://cdn.rawgit.com/nnattawat/flip/master/dist/jquery.flip.min.js"></script>
                        
<div id="example">
    <div class="front">
        <img src="http://community.photostockplus.com/wp-content/uploads/horizontal.jpg"
        width="150px" height="100px">
    </div>
    <div class="back">
        <img src="http://community.photostockplus.com/wp-content/uploads/horizontal.jpg"
        width="150" height="100px">
    </div>
</div>

The top two lines of code here are used in order to connect my code to the server that I learned how to do this from. Once I got that into my code I needed to code an ID and two different classes that represent each side of the "card". When you create the class you will have to name them front and back because the server will be able to identify them. Then inside that div you will add the contents you want on each side. 

                    
<script>
        $("#example").flip({
            axis:'x',
            trigger:'click',
            reverse:true,
        })
</script>

This code here is written in Jquery. This code is telling the HTML above to flip on an X axis, you can change this to flip on the x, y, or even the z axis. The next line is the code that tells the image or text to flip on the trigger on either when it is clicked or when the mouse hovers over the div. The next line which is the reverse line is an optional bit of code that makes the image switch its flipping pattern to a backwards flip.
